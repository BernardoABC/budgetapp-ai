Where should Payee Rules management live?

Payee rules auto-categorize transactions during import. Where should you manage them?

A

Tab inside Import page

Import page gets two tabs: "Import" and "Rules". Rules management is co-located with the feature it powers.

Pros

  • Contextually logical
  • No new nav item

Cons

  • Rules buried inside Import flow
B

Standalone "Rules" page in sidebar nav

A dedicated nav entry between Import and Reports. Full page for rule management.

Pros

  • Easy to find
  • Room to grow

Cons

  • Adds a nav item for a niche feature
C

Modal / drawer from Import page

A "Manage Rules" button on the Import page opens a full-screen modal with the rules table.

Pros

  • No nav change needed
  • Accessible without leaving Import

Cons

  • Slightly hidden
